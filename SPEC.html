EunJod น้องจด

บอท LINE ที่เข้าไปอยู่ใน กลุ่ม LINE แล้วช่วย จดรายรับ-รายจ่าย จากข้อความที่พิมพ์ — แค่พิมพ์ กาแฟ 50 ก็จดเป็นรายจ่าย 50 บาท หมวด กิน > เครื่องดื่ม ให้อัตโนมัติ

▍สรุปสั้น

  • ●แต่ละกลุ่ม = หนึ่งบัญชี (เล่ม) ตั้งด้วย /setbook ชื่อบัญชี ข้อมูลแยกกันสนิท ไม่ปนกัน
  • ●จัดหมวดอัตโนมัติ ด้วยพจนานุกรมคำ + ปัญญาประดิษฐ์ (Claude) ช่วยตอนเจอคำใหม่ และ เรียนรู้ จากที่ผู้ใช้แก้
  • ●รายงานรายเดือน พิมพ์ รายงาน ได้สรุปรายรับ-รายจ่าย-คงเหลือ แยกหมวด + ลิงก์หน้าเว็บที่มีกราฟ
  • ●ฉลาดพอที่จะ เงียบ เมื่อสมาชิกคุยเล่น (ข้อความที่ไม่มีตัวเลขเงิน บอทไม่จด)

😖 ปัญหา

คนอยากรู้ว่า "เงินหายไปไหน" แต่เลิกจดบัญชีกลางคันเพราะ (1) ต้องเปิดแอปแยก (2) ต้องเลือกหมวดเองทุกครั้ง (3) ทำคนเดียว ครอบครัว/ทีมจดร่วมกันไม่ได้ → ไม่มีข้อมูลพอตัดสินใจ และคุมงบไม่ได้จริง

🎯 เป้าหมาย

  1. จดได้ใน 1 ข้อความ ไม่ต้องเลือกหมวด (< 5 วิ)
  2. จัดหมวดถูกเอง > 80% หลังเรียนรู้ ~2 สัปดาห์
  3. แยกบัญชีต่อกลุ่มได้จริง 100%
  4. เห็นภาพการเงินรายเดือนทันที
  5. กลุ่มยังจดต่อหลัง 30 วัน (> 50%)

🚫 ไม่ทำในรอบนี้ (Non-Goals)

• ไม่ทำโปรแกรมบัญชีเต็ม (งบดุล/ภาษี/ใบกำกับ) — นั่นคือ JR ERP
• ไม่รองรับหลายสกุลเงิน (บาทอย่างเดียว)
• ไม่ทำระบบขายหลายลูกค้า/คิดเงิน (ใช้ส่วนตัว)
• ไม่เดาข้อความที่ไม่มีตัวเลขเงิน
• ไม่ทำ OCR สลิป/บิล ใน v1 (เลื่อน Phase 4)
• โฟกัส "กลุ่ม" — แชต 1:1 ไม่ใช่ฟีเจอร์หลัก v1

📝 จดรายการจากข้อความ

P0

  • ✓ กาแฟ 50 → จ่าย 50 · 50 กาแฟ สลับได้
  • ✓ +เงินเดือน 30000 → รับ 30,000
  • ✓ ข้าว 60 #ลูกค้า → โน้ต "ลูกค้า"
  • ✓ หิวจัง (ไม่มีเลข) → ไม่จด

🏗️ สถาปัตยกรรม

  • ▸LINE Messaging API (OA ใหม่ + webhook, ปิด auto-reply, อนุญาตเข้ากลุ่ม)
  • ▸Next.js (App Router) บน Vercel — app/api/line/webhook ตรวจลายเซ็น HMAC-SHA256
  • ▸Supabase (Postgres) — บัญชี/รายการ/หมวด/พจนานุกรม/งบ
  • ▸Claude Haiku จัดหมวดเฉพาะคำใหม่แล้ว cache → ค่า API ต่ำ
  • ▸Vercel Cron → push สรุปสิ้นวัน/เดือน

เลือก stack เดียวกับ JR ERP / TONPALEARN — รีไซเคิลความรู้/แพตเทิร์นเดิม

🗄️ โครงสร้างข้อมูล

  • ledgers — กลุ่ม ↔ บัญชี (line_group_id, name, tz)
  • categories — หมวด 2 ระดับ (parent_id, type)
  • keywords — คำ → หมวด (source: seed/llm/user) ★หัวใจ
  • transactions — รายการ (amount, type, item, note, soft-delete)
  • budgets — งบรายเดือนต่อหมวด
  • report_tokens — ลิงก์เว็บรายงาน (หมดอายุได้)
